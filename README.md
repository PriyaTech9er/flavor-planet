# Flavor Planet: Explore World Cuisine and Culture

---

## 1. What is our topic?

**Flavor Planet** is an interactive web application that allows users to explore the culinary traditions, cultural facts, and music of countries around the world. By spinning a virtual globe, users can discover street foods, unique dining customs, and traditional music from different nations.

---

## 2. Why did we choose this topic?

- **Cultural Awareness:** Food and music are universal languages that connect people. We wanted to create a fun way to learn about global cultures.
- **Engagement:** Interactive exploration (like spinning a globe) makes learning more engaging and memorable.
- **Technology Showcase:** The project demonstrates how modern web frameworks and APIs can be combined to deliver rich, dynamic content.

---

## 3. Frameworks and Technologies Used

### Frontend
- **Next.js** (React framework for SSR and fast development)
- **React-Bootstrap** (UI components)
- **MobX** (state management)
- **react-globe.gl** (3D globe visualization)

### Backend
- **NestJS** (Node.js framework for scalable server-side applications)
- **OpenAI API** (for generating dynamic content about food, culture, and music)
- **Axios** (HTTP client)

---

## 4. Current Implementation

- **Interactive Globe:** Users can spin a 3D globe and select a country.
- **Country Details:** For each selected country, the app displays:
  - A popular street food (with image and description)
  - A unique cultural fact or dining etiquette
  - A traditional song or music, with a YouTube video embedded if available
- **Dynamic Content:** The backend uses OpenAI to generate fresh, relevant content for each country, making the experience unique every time.
- **Modern UI:** The frontend is responsive and visually appealing, using Bootstrap and custom styles.

---

## 5. Future Scope

- **User Accounts:** Allow users to save favorite countries, foods, or songs.
- **More Countries:** Expand the database to include more countries and regions.
- **Deeper Content:** Add recipes, video guides, and local restaurant recommendations.
- **Social Features:** Let users share discoveries or create custom playlists.
- **Localization:** Support multiple languages for a global audience.
- **Mobile App:** Develop a mobile version for on-the-go exploration.
